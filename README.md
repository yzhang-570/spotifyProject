# Spotify Project


## Project Structure
frontend/
- /styles       # CSS files
- /components
- /pages

backend/
- /routes
- /db           # all firebase interactions

## Getting Started
1. Change directories into frontend folder
`cd frontend`  

2. Run react app
```bash
npm install
npm run dev
```

3. Change directories into backend folder in a **different terminal**
`cd backend`  

4. Run express server
```bash
npm install
npm start
```
