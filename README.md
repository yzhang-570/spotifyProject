# Spotify Project


## Project Structure
frontend/
- /styles       # CSS files
- /components
- /pages

backend/
- /routes
- /db           # all firebase interactions

## Getting Started
1. Add firebase API keys to .env in /backend

2. Change directories into frontend folder  
`cd frontend`  

3. Run react app  
```bash
npm install
npm run dev
```

4. Change directories into backend folder in a **different terminal**  
`cd backend`  

5. Run express server  
```bash
npm install
npm start
```
